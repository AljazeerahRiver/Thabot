# أرشيف شركة ثابوت - الصادر والوارد

## نظرة عامة

هذا المستودع يحتوي على نظام أرشفة شامل لجميع المراسلات الصادرة والواردة لشركة ثابوت، ويغطي الشركة والموظفين والعملاء والموردين.

## هيكل الأرشيف

```
أرشيف-ثابوت/
├── 01-الشركة/
│   ├── صادر/ - المراسلات الصادرة من الشركة
│   └── وارد/ - المراسلات الواردة إلى الشركة
├── 02-الموظفون/
│   ├── صادر/ - مراسلات الموظفين الصادرة
│   ├── وارد/ - مراسلات الموظفين الواردة
│   └── قالب-بطاقة-موظف.md - قالب لإنشاء سجل خاص بكل موظف
├── 03-العملاء/
│   ├── صادر/ - المراسلات الصادرة للعملاء
│   ├── وارد/ - المراسلات الواردة من العملاء
│   └── قالب-بطاقة-عميل.md - قالب لإنشاء سجل خاص بكل عميل
└── 04-الموردون/
    ├── صادر/ - المراسلات الصادرة للموردين
    ├── وارد/ - المراسلات الواردة من الموردين
    └── قالب-بطاقة-مورد.md - قالب لإنشاء سجل خاص بكل مورّد
```

## القوالب الفردية

لإنشاء سجل خاص بموظف أو عميل أو مورّد، انسخ القالب المناسب داخل مجلد جديد باسم الجهة:
- الموظفون: `02-الموظفون/[اسم-الموظف]/بطاقة.md`
- - العملاء: `03-العملاء/[اسم-العميل]/بطاقة.md`
  - - الموردون: `04-الموردون/[اسم-المورد]/بطاقة.md`
   
    - ## تصنيف المراسلات وطريقة الترقيم
   
    - | الرمز | النوع | الوصف |
    - |------|------|-------|
    - | ص | صادر | مراسلات صادرة من ثابوت |
    - | و | وارد | مراسلات واردة إلى ثابوت |
    - | صم / وم | موظفون | صادر / وارد للموظفين |
    - | صع / وع | عملاء | صادر / وارد للعملاء |
    - | صمو / ومو | موردون | صادر / وارد للموردين |
   
    - صيغة الترقيم: `[السنة]-[الرمز]-[الرقم التسلسلي]`
    - مثال: `2026-ص-0001` أو `2026-صع-0001`
   
    - ---

    شركة ثابوت - نظام إدارة المراسلات
